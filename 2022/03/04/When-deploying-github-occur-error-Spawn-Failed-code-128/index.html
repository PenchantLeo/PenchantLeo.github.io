When deploying github occur error: Spawn Failed code: 128

Pengcheng Liu
2022-03-04

The day before yesterday I tried to build my own blog with Hexo. Like this:

fatal: Authentication failed for ‘https://github.com/PenchantLeo/PenchantLeo.github.io/': Encountered end of file

FATAL {
    err: Error: Spawn failed
    at ChildProcess.<anonymous> (/Users/apple/Desktop/Blog/node_modules/hexo-util/lib/spawn.js:51:21)
    at ChildProcess.emit (node:events:520:28)
    at Process.ChildProcess._handle.onexit (node:internal/child_process:291:12) {
code: 128
}

} Something’s wrong. Maybe you can find the solution here: %s https://hexo.io/docs/troubleshooting.html

Then I looked up from th internet foa a solution and found some people met the same trouble.

They say there’re something wrong with my ‘_config.yml’ file. I was using the HTTPS as the ‘repo’ stuff. Then I replace the HTTPS link with my SSH links.

(Remember: You need to create and link a SSH to your GitHub account.!!!)

Then open the terminal and input:

  • hexo clean
  • hexo g
  • hexo d

Now deployment is done!
